# 贴吧系统访问说明

## 问题分析

吧广场页面无法显示帖子的原因是：用户可能直接访问了静态的`吧广场.html`文件，而不是Django视图提供的动态页面。

## 解决方案

### 正确的访问方式

请通过以下URL访问吧广场页面：
- **吧广场页面**: http://localhost:8000/tieba-square/
- **我的帖子页面**: http://localhost:8000/profile/
- **首页**: http://localhost:8000/

### 错误的访问方式（避免）

不要直接访问静态HTML文件：
- ❌ `吧广场.html` (静态文件)
- ❌ `个人信息.html` (静态文件)  
- ❌ `首页.html` (静态文件)

## 启动Django服务器

要访问动态页面，需要先启动Django开发服务器：

```bash
cd d:\Django\2023-tieba-bk-20231201086-Lifuli\tieba
python manage.py runserver
```

服务器启动后，访问 http://localhost:8000/tieba-square/ 即可看到动态的吧广场页面。
